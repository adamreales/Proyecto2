Comandas API Backend

api/login
    Descripcion
        Funcion que te permitira desbloquear la funcionalidad a la pagina y dar login al usuario
    post
        Parametros / Request:
            email -> un string de tipo email
            password -> un string de tipo input
        Retorno:
            Si funciona bien
                Te devolvera un json:
                    {
                        "user":
                        "token": "2|QZc5ERF77eSUxnrMRFU3hfWJV3gm7ilOzAExrwzf818fe385"
                    }
                    'status' => 200
            Si da error
                Te devolvera un json:
                    {
                        "error": "Credenciales incorrectas"
                    }
                    'status' => 401
api/registro
    Descripcion
        Funcion que permite dar registro a una usuario nuevo
    post
        Parametros / Request:
            name -> un string de tipo text
            email -> un string de tipo email
            password -> un string de tipo password
            conf_password -> un string de tipo password
        Retorno:
            Si funciona bien
                Te devolvera un json:
                    {
                        "registro": "Registro enviado correctamente. Revisa tu correo",
                        "token": {
                            "accessToken": {
                                "name": "usuario_registro",
                                "abilities": [
                                    "*"
                                ],
                                "expires_at": null,
                                "tokenable_id": 3,
                                "tokenable_type": "App\\Models\\User",
                                "updated_at": "2026-01-27T11:55:56.000000Z",
                                "created_at": "2026-01-27T11:55:56.000000Z",
                                "id": 5
                            },
                            "plainTextToken": "5|j2U7YCS3aupCWIAgdPQQ1Fgek7uIGlBt7HGqFYXj49f2dfbe"
                        }
                    }
            Si da error
                {
                    "error": "Las contraseñas no coinciden"
                }
            
api/perfil
    Descripcion
        Devuelve la informacion del usuario loginado
    get
        Parametros / Request:
            token -> el token del usuario
        Retorno
            Si funciona bien:
                Te devolvera un json
                    "user": {
                        "id": 1,
                        "name": "Adrian",
                        "email": "adrian@test.com",
                        "email_verified_at": "2026-01-27T18:51:50.000000Z",
                        "created_at": "2026-01-27T18:51:50.000000Z",
                        "updated_at": "2026-01-27T18:51:50.000000Z"
                    }
                    'status' => 200
            Si da error
                Te devolvera un json
                    "error": "Error al devolver el usuario"
                    'status' => 401

    Descripcion
